## Getting Started

install dependencies:

```bash
npm i 
```
run the development server :

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

build command :
```bash
npm run build
```

test launch :
```bash
npm run start
```
